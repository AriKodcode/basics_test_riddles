"## This project is a RIDDLE game.
You start the game by entering node app.js in the terminal. Then the game starts. You have several questions with several answers for each question. Until you choose the correct answer, you will not be able to move on to the next question.
At the end of the game, you will be shown a summary of how long the game took you and what the average score for each question is."
